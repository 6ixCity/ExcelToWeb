# ExcelToWeb

### What is ExcelToWeb?
With ExcelToWeb, you can upload your .xlsx files to a
localhost website and view your excel charts in the web.

### How to run?
```
git clone https://github.com/fogleman/Minecraft.git
pip install requirements.py
python main.py
```
